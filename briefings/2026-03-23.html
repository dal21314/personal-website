ANTHROPIC & CLAUDE

Claude Can Now Use Your Mac — Today's Biggest Story
Anthropic announced today that Claude Code and Claude Cowork can now accomplish tasks using your computer — opening files, using the browser, filling spreadsheets, and running dev tools. When enabled, Claude first prioritizes connectors to supported services like Google Workspace or Slack; if no connector is available, it falls back to direct computer control. Available as a research preview for Claude Pro and Max subscribers on macOS. Claude asks for permission before taking significant actions, and Anthropic recommends against using it for sensitive information during this preview phase.
SOURCE // Engadget / 9to5Mac READ →
Claude Code Channels: Text Your Agent from Discord or Telegram
Anthropic launched Claude Code Channels, connecting Claude Code to Discord and Telegram so developers can message their coding agent asynchronously while on the go. The architecture uses MCP as a two-way bridge — spin up a session with the --channels flag and your phone becomes a remote control for your dev environment. Admins can manage channels with an approved sender allow-list for security. This is widely seen as Anthropic's direct answer to the OpenClaw movement.
SOURCE // VentureBeat / Silicon Republic READ →
Cowork Projects Feature Live (Earlier This Week)
Anthropic expanded Claude Cowork Desktop with a Projects feature that keeps files, instructions, and task context organized in a single workspace. Paid users can start fresh, import an existing chat, or connect a local folder for persistent context across sessions — eliminating the context drift that forces users to restate goals repeatedly. Available on Pro, Team, and Enterprise plans.

SECURE MESSAGING SPOTLIGHT

● SIGNAL
IBM + Signal: Building Quantum-Safe Group Messaging from the Ground Up
IBM Research is collaborating with Signal's development team to redesign group messaging protocols to resist future quantum computer attacks — addressing the "harvest now, decrypt later" threat. Rather than simply swapping in quantum-safe algorithms (which would increase bandwidth up to 100x), the team rebuilt the protocol entirely, making group members themselves the gatekeepers instead of the server. Findings were presented at the Real-World Crypto conference this month. Signal already upgraded to the SPQR protocol in 2025 for message content; this work tackles the harder metadata problem — protecting who belongs to which group.
SOURCE // IBM Research / CyberInsider READ →
///
● THREEMA
IBM + Threema: Quantum-Safe Encryption Integration in Progress
Threema is also working with the same IBM Research team to integrate the NIST-standardized ML-KEM algorithm into its messaging architecture — providing a quantum-resilient communication foundation without compromising operational efficiency. Threema CEO Robin Simon called the collaboration foundational for "quantum-secure communication of tomorrow." The work involves exploring how ML-KEM, designed for general encryption like securing communication channels, can be implemented across Threema's global user base while maintaining its strict privacy guarantees.
SOURCE // IBM Research / Quantum Computing Report READ →
// Privacy Note
Threema remains the gold standard for anonymous messaging — no phone number, no email, no server-side account data to subpoena. The IBM quantum-safe research partnership only strengthens that position for the long haul.
